## Concepts fondamentaux

La **programmation orientée objet (POO)** regroupe des données (*attributs*) et des fonctions (*méthodes*) dans une *classe*. À partir d'une classe, on crée des *instances* (objets).

Tu utilises déjà des objets sans le savoir :

```python
from machine import Pin
from neopixel import NeoPixel

led = Pin(2, Pin.OUT)       # led est un objet de la classe Pin
np = NeoPixel(Pin(26), 8)   # np est un objet de la classe NeoPixel
np.write()                  # write() est une méthode de l'objet np
```

### Définir une première classe

```python
class Voiture:
    marque = "Renault"
    modele = "Clio"
    annee = 2020

ma_voiture = Voiture()
print(ma_voiture.marque)    # Renault
```

### Le constructeur `__init__`

Le constructeur `__init__(self)` est appelé automatiquement à la création de l'objet. `self` est la référence à l'objet en cours de création.

```python
class Voiture:
    def __init__(self):
        self.marque = "Renault"
        self.modele = "Clio"
        self.annee = 2020

ma_voiture = Voiture()
print(ma_voiture.marque)    # Renault
```

### Paramètres au constructeur

```python
class Voiture:
    def __init__(self, marque, modele, annee):
        self.marque = marque
        self.modele = modele
        self.annee = annee

    def description(self):
        return self.marque + " " + self.modele + " (" + str(self.annee) + ")"

voiture1 = Voiture("Renault", "Clio", 2020)
voiture2 = Voiture("Peugeot", "208", 2022)

print(voiture1.description())   # Renault Clio (2020)
print(voiture2.description())   # Peugeot 208 (2022)
```

## Application sur ESP32

On peut créer une classe `LED` pour encapsuler la logique de clignotement :

```python
from machine import Pin
import time

class LED:
    def __init__(self, pin_num):
        self.pin = Pin(pin_num, Pin.OUT)

    def allumer(self):
        self.pin.on()

    def eteindre(self):
        self.pin.off()

    def basculer(self):
        self.pin.value(not self.pin.value())

    def clignoter(self, nb_fois, delai_ms=200):
        for _ in range(nb_fois):
            self.allumer()
            time.sleep_ms(delai_ms)
            self.eteindre()
            time.sleep_ms(delai_ms)

# Utilisation
led_rouge = LED(2)
led_verte = LED(19)

led_rouge.clignoter(3)
led_verte.clignoter(5, delai_ms=100)
```

## Héritage

L'**héritage** permet à une classe enfant de réutiliser et d'étendre les méthodes d'une classe parente.

```python
class Animal:
    def __init__(self, nom):
        self.nom = nom

    def parler(self):
        return "..."

class Chien(Animal):
    def parler(self):
        return "Woof !"

class Chat(Animal):
    def parler(self):
        return "Miaou !"

chien = Chien("Rex")
chat  = Chat("Félix")

print(chien.nom + " dit : " + chien.parler())   # Rex dit : Woof !
print(chat.nom  + " dit : " + chat.parler())    # Félix dit : Miaou !
```

## Algorithme PGCD

La **récursivité** est un concept fondamental : une fonction qui s'appelle elle-même. L'algorithme d'Euclide pour le PGCD (Plus Grand Commun Diviseur) en est un exemple classique.

```python
def pgcd(a, b):
    if b == 0:
        return a
    return pgcd(b, a % b)

print(pgcd(48, 18))    # 6
print(pgcd(100, 75))   # 25
```

  <h4 class="font-bold text-blue-400 mb-2">💡 Comment ça marche</h4>
  <p class="text-sm text-base-content/70">pgcd(48, 18) → pgcd(18, 12) → pgcd(12, 6) → pgcd(6, 0) → retourne 6. La condition <code>if b == 0</code> est le cas de base qui arrête la récursion.</p>
