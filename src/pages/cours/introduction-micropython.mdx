## Qu'est-ce que MicroPython ?

MicroPython est une implémentation légère de Python 3, conçue pour tourner sur des microcontrôleurs avec peu de mémoire. Créé en 2013 par Damien George, maintenu par la communauté open source.

### Python vs MicroPython

## Les modules essentiels

### `machine` — contrôle du matériel

```python
from machine import Pin, PWM, ADC

# GPIO digital
led = Pin(2, Pin.OUT)
led.on()   # Allumer
led.off()  # Éteindre

# Bouton avec pull-up
btn = Pin(0, Pin.IN, Pin.PULL_UP)
print(btn.value())  # 0 = appuyé

# PWM (signal variable — pour servos, luminosité)
pwm = PWM(Pin(2))
pwm.freq(1000)   # 1 kHz
pwm.duty(512)    # 50% (plage 0-1023)
```

### `network` — WiFi

```python
import network

# Mode AP — l'ESP32 crée son propre réseau
ap = network.WLAN(network.AP_IF)
ap.active(True)
ap.config(essid='MonESP32', password='12345678')

# Mode STA — rejoindre un réseau existant
sta = network.WLAN(network.STA_IF)
sta.active(True)
sta.connect('MonWiFi', 'motdepasse')
while not sta.isconnected():
    pass
print('IP:', sta.ifconfig()[0])
```

### `time` — temporisation

```python
import time

time.sleep(1)        # Pause 1 seconde
time.sleep_ms(500)   # Pause 500 ms
t = time.ticks_ms()  # Compteur en ms
```

## Le REPL — ton meilleur outil

Le **REPL** est une console interactive directe sur l'ESP32. Connecte-toi avec Thonny et tape :

```python
>>> from machine import Pin
>>> led = Pin(2, Pin.OUT)
>>> led.on()    # La LED s'allume instantanément !
>>> led.off()
```

Pas besoin de compiler ni de rebooter — idéal pour tester et apprendre.
