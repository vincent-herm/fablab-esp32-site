## Générer un son avec le PWM

Un haut-parleur ou buzzer connecté sur une broche PWM vibre à la fréquence du signal généré. Si la fréquence est entre **20 Hz et 20 000 Hz**, le son est audible.

```python
from machine import Pin, PWM
import time

hp = PWM(Pin(5))
hp.duty(40)         # rapport cyclique ~4% (suffit pour un son correct)
hp.freq(1000)       # fréquence : 1000 Hz
time.sleep(1)
hp.deinit()         # arrêter le PWM et libérer la broche
```

<div class="rounded-xl border border-blue-500/20 bg-blue-500/5 p-5 my-6">
  <h4 class="font-bold text-blue-400 mb-2">💡 Rapport cyclique faible</h4>
  <p class="text-sm text-base-content/70">Un rapport cyclique de ~4% est volontairement faible : la membrane est poussée brièvement, puis relâchée longtemps. C'est suffisant pour produire le son sans risquer de surchauffe.</p>
</div>

### Faire varier la fréquence

```python
from machine import Pin, PWM
import time

hp = PWM(Pin(5))
hp.duty(40)

# Balayage de 200 Hz à 2000 Hz
for freq in range(200, 2001, 50):
    hp.freq(freq)
    time.sleep_ms(50)

hp.deinit()
```

## Notes musicales — la gamme tempérée

En musique occidentale, le **La3 (A4) = 440 Hz** est la fréquence de référence (le diapason). Les 12 notes d'une octave forment une *suite géométrique* : chaque demi-ton multiplie la fréquence par **2^(1/12) ≈ 1,0595**.

Après 12 demi-tons : fréquence × 2 = octave supérieure. La4 = 880 Hz, La5 = 1760 Hz…

```python
# Générer les 13 notes d'une octave (La4 = 880 Hz)
liste_freq = [int(2**(x/12)*880) for x in range(0, 13)]
# [880, 932, 988, 1047, 1109, 1175, 1245, 1319, 1397, 1480, 1568, 1661, 1760]
```

### Jouer une gamme

```python
from machine import Pin, PWM
import time

hp = PWM(Pin(5))
hp.duty(40)

# La4 = 880 Hz — 13 notes d'une octave
notes = [int(2**(x/12)*880) for x in range(0, 13)]

for freq in notes:
    hp.freq(freq)
    time.sleep_ms(200)

hp.deinit()
```

### Les gammes musicales

Une **gamme** est une sélection de 7 notes dans l'octave. Chaque gamme produit une couleur émotionnelle différente :

<div class="overflow-x-auto rounded-xl border border-base-200/80 my-6">
  <table class="table w-full">
    <thead>
      <tr><th>Gamme</th><th>Intervalles</th><th>Couleur</th></tr>
    </thead>
    <tbody>
      <tr><td class="font-medium">Majeure</td><td class="font-mono text-xs">[0,2,4,5,7,9,11,12]</td><td>Joyeuse</td></tr>
      <tr><td class="font-medium">Éolienne</td><td class="font-mono text-xs">[0,2,3,5,7,8,10,12]</td><td>Mélancolique</td></tr>
      <tr><td class="font-medium">Lydienne</td><td class="font-mono text-xs">[0,2,4,6,7,9,11,12]</td><td>Aérienne</td></tr>
      <tr><td class="font-medium">Dorienne</td><td class="font-mono text-xs">[0,2,3,5,7,9,10,12]</td><td>Jazzy</td></tr>
    </tbody>
  </table>
</div>

```python
from machine import Pin, PWM
import time

hp = PWM(Pin(5))
hp.duty(40)

base = 880  # La4
tous_demitons = [int(2**(x/12)*base) for x in range(13)]

# Gamme majeure
majeure = [0, 2, 4, 5, 7, 9, 11, 12]
for i in majeure:
    hp.freq(tous_demitons[i])
    time.sleep_ms(300)

hp.deinit()
```

  <p class="text-sm text-base-content/70">Le <strong>swing</strong> alterne des durées longues et courtes entre les notes, créant un effet rythmique. Par exemple, alterner 300 ms et 150 ms entre chaque note transforme une gamme monotone en mélodie vivante.</p>
