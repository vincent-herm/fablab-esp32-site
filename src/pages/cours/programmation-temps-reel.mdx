## Le problème du code bloquant

Avec `time.sleep()`, ton programme est **bloqué** — il ne peut rien faire d'autre pendant la pause. C'est acceptable pour des projets simples, mais impossible dès qu'on veut réagir à plusieurs événements.

```python
# ❌ Code bloquant — pendant 500 ms, rien d'autre ne peut se passer
while True:
    led.on()
    time.sleep_ms(500)   # bloqué ici
    led.off()
    time.sleep_ms(500)   # bloqué ici aussi
```

---

## Mesure du temps avec `ticks_ms()`

La solution : mémoriser l'heure du dernier événement et **vérifier** si le délai est écoulé sans attendre.

```python
import time

t0 = time.ticks_ms()   # heure de départ (millisecondes)

# ... faire d'autres choses ...

elapsed = time.ticks_diff(time.ticks_ms(), t0)
print(f"Temps écoulé : {elapsed} ms")
```

<div class="rounded-xl border border-blue-500/20 bg-blue-500/5 p-5 my-6">
  <h4 class="font-bold text-blue-400 mb-2">💡 Pourquoi ticks_diff() ?</h4>
  <p class="text-sm text-base-content/70">
    Le compteur <code>ticks_ms()</code> déborde après environ 12 jours. <code>ticks_diff()</code> gère correctement ce débordement — ne jamais faire <code>t2 - t1</code> directement.
  </p>
</div>

## Boucle rapide non-bloquante

La technique fondamentale : une boucle qui tourne très vite (1-10 ms), et qui vérifie si c'est le bon moment pour faire chaque action.

```python
from machine import Pin
import time

led  = Pin(2, Pin.OUT)
bp   = Pin(0, Pin.IN, Pin.PULL_UP)

# Mémoriser les derniers états
t_led   = time.ticks_ms()
etat    = False

while True:
    maintenant = time.ticks_ms()

    # Tâche 1 : clignoter la LED toutes les 500 ms
    if time.ticks_diff(maintenant, t_led) >= 500:
        etat = not etat
        led.value(etat)
        t_led = maintenant   # réinitialiser le chrono

    # Tâche 2 : lire le bouton (sans bloquer)
    if bp.value() == 0:
        print("Bouton appuyé !")

    time.sleep_ms(10)   # cycle de 10 ms
```

### Plusieurs tâches indépendantes

```python
from machine import Pin
from neopixel import NeoPixel
import time

led  = Pin(2, Pin.OUT)
np   = NeoPixel(Pin(26, Pin.OUT), 4)
bp   = Pin(0, Pin.IN, Pin.PULL_UP)

t_led  = time.ticks_ms()
t_np   = time.ticks_ms()
etat   = False
couleur = 0

while True:
    now = time.ticks_ms()

    # Tâche 1 : LED clignote à 2 Hz (250 ms)
    if time.ticks_diff(now, t_led) >= 250:
        etat = not etat
        led.value(etat)
        t_led = now

    # Tâche 2 : NeoPixel change de couleur toutes les 2 s
    if time.ticks_diff(now, t_np) >= 2000:
        couleurs = [(255,0,0), (0,255,0), (0,0,255), (255,255,0)]
        for i in range(4):
            np[i] = couleurs[couleur]
        np.write()
        couleur = (couleur + 1) % len(couleurs)
        t_np = now

    # Tâche 3 : lire le bouton en permanence
    if bp.value() == 0:
        print("Bouton !")

    time.sleep_ms(10)
```

## Séquenceur — machine à états

Pour des séquences plus complexes, on utilise une **machine à états**. Le programme mémorise dans quel "état" il se trouve et quand passer au suivant.

```python
from machine import Pin
import time

led_rouge  = Pin(23, Pin.OUT)
led_verte  = Pin(19, Pin.OUT)
led_bleue  = Pin(18, Pin.OUT)

# Séquence : rouge 1s → vert 2s → bleu 500ms → retour
sequence = [
    (led_rouge, 1000),
    (led_verte, 2000),
    (led_bleue,  500),
]

etat = 0
t0   = time.ticks_ms()

# Allumer la première LED
for led, _ in sequence:
    led.off()
sequence[etat][0].on()

while True:
    now = time.ticks_ms()
    duree_actuelle = sequence[etat][1]

    if time.ticks_diff(now, t0) >= duree_actuelle:
        # Éteindre l'état actuel
        sequence[etat][0].off()
        # Passer à l'état suivant
        etat = (etat + 1) % len(sequence)
        sequence[etat][0].on()
        t0 = now

    time.sleep_ms(10)
```

## Minuterie d'éclairage

Exemple pratique : LED qui s'allume quand on appuie, et s'éteint après 3 secondes.

```python
from machine import Pin
import time

led = Pin(2, Pin.OUT)
bp  = Pin(0, Pin.IN, Pin.PULL_UP)

DUREE_MS = 3000   # 3 secondes
t_allumage = None

while True:
    now = time.ticks_ms()

    # Bouton appuyé → allumer et mémoriser l'heure
    if bp.value() == 0:
        led.on()
        t_allumage = now

    # Délai écoulé → éteindre
    if t_allumage is not None:
        if time.ticks_diff(now, t_allumage) >= DUREE_MS:
            led.off()
            t_allumage = None

    time.sleep_ms(10)
```

## Interruptions

Les interruptions permettent de réagir **immédiatement** à un événement (front montant ou descendant d'un pin), sans attendre le prochain tour de boucle.

```python
from machine import Pin
import time

led = Pin(2, Pin.OUT)

# Variable partagée entre l'interruption et la boucle principale
evenement = False

def handler(pin):
    global evenement
    evenement = True

bp = Pin(0, Pin.IN, Pin.PULL_UP)
bp.irq(trigger=Pin.IRQ_FALLING, handler=handler)   # front descendant

while True:
    if evenement:
        led.toggle()
        evenement = False
        print("Interruption détectée !")
    time.sleep_ms(10)
```

### Mesure du temps entre deux fronts

```python
from machine import Pin
import time

t_debut = None
t_fin   = None

def front_montant(pin):
    global t_debut
    t_debut = time.ticks_ms()

def front_descendant(pin):
    global t_fin
    t_fin = time.ticks_ms()

bp = Pin(0, Pin.IN, Pin.PULL_UP)
bp.irq(trigger=Pin.IRQ_RISING,  handler=front_montant)
bp.irq(trigger=Pin.IRQ_FALLING, handler=front_descendant)

while True:
    if t_debut and t_fin:
        duree = time.ticks_diff(t_fin, t_debut)
        print(f"Durée appui : {duree} ms")
        t_debut = t_fin = None
    time.sleep_ms(10)
```
