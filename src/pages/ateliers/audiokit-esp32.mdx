## Partie 2 — Codec ES8388 via I2S

Le protocole **I2S** (Inter-IC Sound) est conçu spécifiquement pour l'audio numérique. Il sépare les données audio de l'horloge sur des fils dédiés.

### Architecture I2S sur l'AudioKit

<div class="rounded-xl overflow-hidden border border-base-200/60 bg-[#1a1b26] my-4">
  <div class="px-4 py-2 border-b border-white/8 text-white/50 text-xs font-mono">connexions I2S</div>
  <pre class="p-4 text-xs font-mono text-white/70 leading-relaxed overflow-x-auto"><code>{`ESP32          ES8388
GPIO 27  ────  BCLK   (Bit Clock)
GPIO 25  ────  WCLK   (Word Clock / LR Select)
GPIO 26  ────  DOUT   (Data Out → vers codec)
GPIO 35  ────  DIN    (Data In  ← depuis codec)

GPIO 32  ────  SDA    (I2C - configuration)
GPIO 33  ────  SCL    (I2C - configuration)`}</code></pre>
</div>

## Étapes de réalisation

### Étape 1 — Flasher MicroPython

Télécharge le firmware ESP32 depuis [micropython.org/download/esp32/](https://micropython.org/download/esp32/) :

<div class="rounded-xl overflow-hidden border border-base-200/60 bg-[#1a1b26] my-4">
  <div class="px-4 py-2 border-b border-white/8 text-white/50 text-xs font-mono">terminal</div>
  <pre class="p-4 text-xs font-mono text-white/70 leading-relaxed overflow-x-auto"><code>{`esptool.py --port /dev/ttyUSB0 erase_flash
esptool.py --port /dev/ttyUSB0 --baud 460800 write_flash -z 0x1000 esp32-firmware.bin`}</code></pre>
</div>

### Étape 2 — Connecter le haut-parleur

Connecte le haut-parleur au connecteur **SPK** sur la carte (polaire rouge sur le `+`).

### Étape 3 — Copier le code

<div class="rounded-xl overflow-hidden border border-base-200/60 bg-[#1a1b26] my-4">
  <div class="px-4 py-2 border-b border-white/8 text-white/50 text-xs font-mono">terminal</div>
  <pre class="p-4 text-xs font-mono text-white/70 leading-relaxed overflow-x-auto"><code>{`mpremote cp main.py :main.py && mpremote reset`}</code></pre>
</div>

### Étape 4 — Tester

Appuie sur les boutons de la carte pour :
- **BTN1** : jouer la gamme Do → Do
- **BTN2** : monter le volume
- **BTN3** : baisser le volume
- **BTN4** : jouer une mélodie
