## Ce que tu vas créer

Un système d'escape game complet :
- Un **lecteur RFID** qui reconnaît les badges autorisés
- Un **verrou** (LED verte/rouge) qui s'ouvre avec le bon badge
- Un **buzzer** qui s'active en cas de badge refusé
- Une **logique de jeu** : 3 badges à trouver pour déverrouiller le coffre final

---

## Le protocole RFID et le RC522

Le **RC522** est un module lecteur RFID 13.56 MHz très répandu. Il communique avec l'ESP32 via **SPI**.

## Concepts clés

### Lire un badge RFID

```python
from machine import Pin, SPI
import mfrc522

spi = SPI(1, baudrate=1_000_000,
          sck=Pin(18), mosi=Pin(23), miso=Pin(19))
rfid = mfrc522.MFRC522(spi, gpioRst=22, gpioCs=5)

while True:
    stat, tag_type = rfid.request(rfid.REQIDL)
    if stat == rfid.OK:
        stat, uid = rfid.anticoll()
        if stat == rfid.OK:
            uid_str = ':'.join([f'{b:02X}' for b in uid])
            print(f"Badge détecté : {uid_str}")
```

### Système d'autorisation

```python
# UID des badges autorisés (lire avec le script d'enrôlement)
BADGES_AUTORISES = {
    'AB:12:34:56': 'Badge rouge',
    'CD:78:9A:BC': 'Badge bleu',
    'EF:01:23:45': 'Badge maître',
}

def verifier_badge(uid):
    uid_str = ':'.join([f'{b:02X}' for b in uid])
    if uid_str in BADGES_AUTORISES:
        nom = BADGES_AUTORISES[uid_str]
        print(f"✓ Accès accordé : {nom}")
        return True
    else:
        print(f"✗ Badge inconnu : {uid_str}")
        return False
```

### Logique de jeu — 3 badges requis

```python
from machine import Pin
import time

led_verte = Pin(2, Pin.OUT)
led_rouge = Pin(4, Pin.OUT)
buzzer    = Pin(15, Pin.OUT)

BADGES_REQUIS = {'AB:12:34:56', 'CD:78:9A:BC', 'EF:01:23:45'}
badges_trouves = set()

def badge_scanne(uid_str):
    if uid_str in BADGES_REQUIS:
        badges_trouves.add(uid_str)
        led_verte.on()
        time.sleep_ms(500)
        led_verte.off()

        if badges_trouves == BADGES_REQUIS:
            victoire()
    else:
        # Badge non autorisé : alarme
        for _ in range(3):
            led_rouge.on()
            buzzer.on()
            time.sleep_ms(200)
            led_rouge.off()
            buzzer.off()
            time.sleep_ms(100)

def victoire():
    print("🎉 COFFRE OUVERT !")
    for _ in range(10):
        led_verte.toggle()
        time.sleep_ms(100)
```
