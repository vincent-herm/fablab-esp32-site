## Ce que tu vas créer

Un mini-clavier musical avec deux modes :
- **Mode libre** : chaque bouton joue une note (Do, Ré, Mi, Sol) tant qu'il est appuyé
- **Mode mélodie** : l'ESP32 joue Frère Jacques automatiquement en boucle
- Le bouton BOOT sur GPIO0 bascule entre les deux modes

## Comment ça marche

### Dictionnaire de notes

Chaque note musicale correspond à une fréquence précise en Hertz. On stocke tout ça dans un dictionnaire :

```python
NOTES = {
    'Do':  262,
    'Re':  294,
    'Mi':  330,
    'Fa':  349,
    'Sol': 392,
    'La':  440,
    'Si':  494,
    'Do2': 523,
}
```

### La fonction jouer_note

Le buzzer passif est contrôlé par PWM : on lui envoie un signal à la fréquence voulue, et il vibre pour produire le son correspondant.

```python
from machine import PWM, Pin

buzzer = PWM(Pin(5))

def jouer_note(freq, duree_ms=200):
    if freq == 0:
        buzzer.duty(0)      # silence
    else:
        buzzer.freq(freq)
        buzzer.duty(512)    # 50% duty cycle = volume max
    time.sleep_ms(duree_ms)
    buzzer.duty(0)          # couper le son

# Jouer Do
jouer_note(NOTES['Do'], 300)
```
