## Ce que tu vas créer

Un système de contrôle d'accès par badge RFID :
- **Mode enrôlement** : présente un badge pour l'enregistrer dans la liste autorisée
- **Mode lecture** : chaque badge présenté est comparé à la liste
- Accès accordé → LED verte + bip court
- Badge refusé → LED rouge + alarme

## Comment ça marche

### Convertir l'UID en chaîne lisible

Chaque carte RFID possède un identifiant unique (UID) de 4 à 7 octets. On le convertit en chaîne hexadécimale pour le stocker et le comparer :

```python
def uid_vers_str(uid):
    """Convertit une liste d'octets en chaîne hex lisible."""
    return ':'.join('{:02X}'.format(b) for b in uid)

# Exemple : [0xDE, 0xAD, 0xBE, 0xEF] → "DE:AD:BE:EF"
```

### La fonction d'enrôlement

Le mode enrôlement permet d'ajouter facilement de nouveaux badges sans modifier le code :

```python
badges_autorises = []

def enrolement(rdr):
    """Attend un badge et l'ajoute à la liste autorisée."""
    print("Présente le badge à enrôler...")
    while True:
        stat, tag_type = rdr.request(rdr.REQIDL)
        if stat == rdr.OK:
            stat, uid = rdr.anticoll()
            if stat == rdr.OK:
                uid_str = uid_vers_str(uid)
                if uid_str not in badges_autorises:
                    badges_autorises.append(uid_str)
                    print(f"Badge enrôlé : {uid_str}")
                else:
                    print(f"Badge déjà enrôlé : {uid_str}")
                return uid_str
```
