## Ce que tu vas créer

Un système de contrôle d'accès RFID complet avec :
- **3 niveaux** : Visiteur (1), Membre (2), Admin (3)
- **Phase de programmation** : scanner un badge, saisir nom + prénom, choisir le niveau
- **Phase de production** : scanner → LED bleue si accès accordé, rien si refusé
- **Sauvegarde persistante** : les badges sont stockés en JSON sur l'ESP32

## Comment ça marche

### Le menu principal

Au lancement, le Shell affiche un menu :
- **1. Programmation** : enregistrer des badges (nom + niveau)
- **2-4. Production** : contrôle d'accès avec niveau minimum requis
- **5. Lister** : voir tous les badges enregistrés
- **6. Supprimer** : retirer un badge

### Les 3 niveaux

En mode production, on choisit le niveau minimum requis. Un badge de niveau 3 (Admin) passe partout, un badge de niveau 1 (Visiteur) ne passe que si le niveau requis est 1.

### Sauvegarde en JSON

Les badges sont stockés dans `badges.json` sur l'ESP32. Le fichier persiste après un redémarrage — pas besoin de reprogrammer les badges à chaque fois.
