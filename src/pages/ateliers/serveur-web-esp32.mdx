## Ce que tu vas créer

À la fin de cet atelier, ton ESP32 sera un **mini-serveur web** qui :
- Crée son propre réseau WiFi (pas besoin de box)
- Sert une page HTML avec des boutons ON/OFF
- Contrôle une LED physique connectée au GPIO2

## Étapes de réalisation

### Étape 1 — Flasher MicroPython

Télécharge le firmware depuis [micropython.org/download/esp32/](https://micropython.org/download/esp32/) et flash-le :

<div class="rounded-xl overflow-hidden border border-base-200/60 bg-[#1a1b26] my-4">
  <div class="px-4 py-2 border-b border-white/8 text-white/50 text-xs font-mono">terminal</div>
  <pre class="p-4 text-xs font-mono text-white/70 leading-relaxed overflow-x-auto"><code>{`esptool.py --port /dev/ttyUSB0 erase_flash
esptool.py --port /dev/ttyUSB0 --baud 460800 write_flash -z 0x1000 esp32-firmware.bin`}</code></pre>
</div>

### Étape 2 — Câbler la LED

GPIO2 → résistance 220Ω → LED (anode) → LED (cathode) → GND

### Étape 3 — Copier le code

<div class="rounded-xl overflow-hidden border border-base-200/60 bg-[#1a1b26] my-4">
  <div class="px-4 py-2 border-b border-white/8 text-white/50 text-xs font-mono">terminal</div>
  <pre class="p-4 text-xs font-mono text-white/70 leading-relaxed overflow-x-auto"><code>{`mpremote cp main.py :main.py && mpremote reset`}</code></pre>
</div>

### Étape 4 — Tester

1. Connecte-toi au WiFi `ESP32-Fablab` (mot de passe : `micropython`)
2. Ouvre `http://192.168.4.1` dans ton navigateur
3. Clique sur les boutons pour contrôler la LED !
